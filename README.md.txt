# W-PCs Enterprise Network

## Project Overview

This repository documents the design and implementation of the network infrastructure for W-PCs Ltd, a fictional computer sales and IT support company.

The aim of this project is to simulate the planning, deployment, configuration and ongoing expansion of a real enterprise network while developing practical networking and cybersecurity skills.

This project is being built using Cisco Packet Tracer and will continue to evolve over time as new technologies and services are introduced.

---

## Objectives

- Design a scalable enterprise network
- Configure routers and switches
- Implement IPv4 addressing
- Configure DHCP and DNS
- Secure wireless networking
- Implement VLANs
- Configure routing protocols
- Apply Access Control Lists (ACLs)
- Document every stage of development
- Build a professional networking portfolio

---
