# W-PCs Enterprise Network

## Project Overview

This repository documents the design, implementation and ongoing expansion of the network infrastructure for **W-PCs Ltd**, a fictional computer sales and IT support company.

The aim of this project is to simulate the planning, deployment, configuration and maintenance of a real enterprise network while developing practical networking, systems administration and cybersecurity skills.

The project is being built using **Cisco Packet Tracer** and will continue to evolve through multiple phases into a fully featured enterprise environment incorporating routing, switching, VLANs, Windows Server, Active Directory, network security and penetration testing.
